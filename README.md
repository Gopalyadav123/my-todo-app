# 📝 React Todo App (Context API)

A simple and clean Todo Application built using **React.js** and **Context API** for global state management.



## 🚀 Features

* ➕ Add new tasks
* 📋 Display task list in table format
* 🌐 Global state using Context API
* ⚡ Fast performance using Vite
* 🎯 Beginner-friendly project structure



## 🛠️ Tech Stack

* ⚛️ React.js
* 📦 Context API
* 💻 JavaScript (ES6+)
* ⚡ Vite
* 🎨 CSS



## 📂 Project Structure


src/
│── App.jsx
│── TodoContext.jsx
│── AddTodoForm.jsx
│── Todoitem.jsx
│── Todoitemlist.jsx
│── App.css


## ▶️ Run Locally

bash

npm run dev
